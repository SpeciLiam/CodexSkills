# LinkedIn Batch Drain Codex

Use the `linkedin-batch-drain-codex` skill
(`skills/linkedin-batch-drain-codex/SKILL.md`).

20 fresh non-duplicate LinkedIn early-career SWE postings, then tailor verified
one-page resumes efficiently with no-browser workers where safe, and finally
apply one browser form at a time in ATS-friction order.

browser work, workers, or monitor loops. Submit high-confidence applications
when the tailored resume is verified, required answers are truthful or covered
by standing answers, and no true blocker remains. Park only true blockers with
precise manual handoffs.
