# Peer Handoff

Use `handoffs/README.md` and create or update a handoff for: $ARGUMENTS

The handoff must include:

- sender and receiver
- intent
- branch or workspace context
- touched files
- out-of-scope files
- commands already run
- exact questions for the other agent
- expected output format

Keep the handoff short enough that it can be pasted into the other agent directly.
